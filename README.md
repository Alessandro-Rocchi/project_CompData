# Project CompData

To install the required dependencies and make everything work with uv, run the following command:

```In the repository where you want to run the project via terminal, run the following command: **git clone: [text](https://github.com/Alessandro-Rocchi/project_CompData.git)**
```

```After cloning the repository, navigate to the project directory and run the following command to install the dependencies: **uv run main.py**
```

This will install all the necessary dependencies and run the main.py file, allowing you to use the project as intended. Make sure you have uv installed on your system before running the command.

**Have fun!**
